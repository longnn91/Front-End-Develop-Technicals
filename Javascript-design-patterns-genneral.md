## Constructor Pattern
**1. Object Creation**
- Ba cách phổ biến để tạo các đối tượng mới trong JavaScript như sau:

```javascript
// Each of the following options will create a new empty object:
 
var newObject = {};
 
// or
var newObject = Object.create( Object.prototype );
 
// or
var newObject = new Object();
```
